# TMUX

## install [tmux](https://github.com/tmux/tmux/wiki/Installing)

``` bash
sudo dnf install -y tmux
```

## install [tmux plugin manager](https://github.com/tmux-plugins/tpm)

``` bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

## copy config files

- `.tmux.conf` --> `~/`

- `themes` --> `~/.tmux`
